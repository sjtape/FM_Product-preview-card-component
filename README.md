### What I learned

I learn how to change image without using media query.
and css ```clamp()``` .
```html
<
<figure class="product-image">
    <picture>
        <source
            media="(min-width:769px)"
            srcset="./images/image-product-desktop.jpg"
        />
        <img src="./images/image-product-mobile.jpg" alt="Perfume image" />
    </picture>
</figure>
```
